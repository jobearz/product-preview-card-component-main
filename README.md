### What I learned

I learned a lot about flex in terms of designing layouts with both the column and row directions. I also learned a lot about how responsive design works, and how design choices must change based on the screen size you are designing for.

### Continued development

I want to become more comfortable using flexbox. I still feel a bit confused on how the flex properties work and which ones are the best to use depending on the scenario.

### Useful resources

- [Responsive images](https://developer.mozilla.org/en-US/docs/Web/HTML/Guides/Responsive_images) - This helped me understand how to display the images in a responsive design.
